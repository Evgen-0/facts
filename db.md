# Схема бази даних

### Users

| Поле     | Тип          | null | Опис               |
|----------|--------------|------|--------------------|
| id       | uuid         |      | Ідентифікатор      |
| name     | varchar(20)  |      | Унікальне ім'я     |
| email    | varchar(255) |      | Електрона скринька |
| password | varchar(255) |      | Пароль             |
| photo    | varchar(128) | +    | Аватарка           |
